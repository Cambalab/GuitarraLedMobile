
# Pre-install
* node
* npm

# Install

```
git clone http://lanus.elarteylatecnologia.com.ar:8081/fmmainere/guitarraledmobile.git
```

## App Dedepencies

```
npm install
bower install
grunt platform:add:android
```

## Run test
```
grunt test
```

## Run App
```
grunt cordova
grunt run
```

## Run App
```
grunt cordova
adb install -r platforms/android/ant-build/GuitarraLedMobile-debug.apk
```
